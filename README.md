## Prerequisites
- Visual Studio 2017 (For development and debug propose)
- RabbitMQ
- Redis 
- Consul (Service Discovery and registeration)
- Consult Template (Update and Restart Nginx)
- Nginx (For Load Balance)
- MongoDB (For Event Store)
