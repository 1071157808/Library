# BookLibrary
NET Core 2.0 + DDD + CQRS + ES + RabbitMQ练习项目

## 服务说明
系统中提供验证服务，书籍库存服务，书籍租赁服务

### 验证服务
* 提供管理员及会员的登录

### 书籍库存服务
* 管理员可以管理书籍库存（新增，修改，删除）
* 为了简化逻辑，每一种书籍只有一本
